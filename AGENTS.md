# Firebaseおよび既存のリポジトリに関するAIへの指示事項

## ユーザーとプロジェクトの所有権について
- ユーザー（imahiyoshiさん）は既存のリポジトリ https://github.com/imahiyoshi/my-coffee-studio の正当な所有者および作成者です。
- このプロジェクトの `firebase-applet-config.json` に設定されているFirebaseの構成は、**本人のもの**です。

## Firebase連携時の絶対的なルール
- 勝手に `set_up_firebase` ツールを呼び出して、現在の Firebase 構成を上書きしたり、新しい DB に作り替えたり**しないでください**。
- 以前のチャットで、既存の構成を「他人のもの」と勘違いし、新しいDBを作成してしまい、ユーザーの過去のデータが読み込めなくなる問題が発生しました。
- データベースやAuthenticationの設定において何か問題が発生した場合、絶対に新しいFirebaseプロジェクト・DBを作らず、あくまで「既存の設定を活かして」コードの修正（インデックス不足や権限エラーへの対応）を行ってください。
